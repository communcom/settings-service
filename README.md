# SETTINGS-SERVICE

#### Clone the repository

```bash
git clone https://github.com/communcom/settings-service.git
cd settings-service
```

#### Create .env file

```bash
cp .env.example .env
```

#### Create docker-compose file

```bash
cp docker-compose.example.yml docker-compose.yml 
```

#### Run

```bash
docker-compose up -d --build
```
